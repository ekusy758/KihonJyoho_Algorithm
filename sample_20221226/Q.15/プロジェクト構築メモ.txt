【開発】
-----------------------------------------------------------
①npm create vite@latestを実行、Viteプロジェクトを開始
  >> Project name: プロジェクト名を入力

  >> Select a framework: Reactを選択

  >> Select a variant: JavaScriptを選択

②以下のコマンドを実行
  cd <プロジェクト名>
  npm install

③npm run devを実行し、ローカルサーバを起動
  http://localhost:5173/へアクセス

-----------------------------------------------------------
【ビルド】
-----------------------------------------------------------
①npm run buildを実行し、dist/フォルダを確認

②npm run previewし、ローカルサーバを起動
  http://localhost:4173/へアクセス
